What is an API Key?

An API key is a unique identifier that allows this extension to securely communicate with Google's Gemini AI service to generate video summaries.

What it's used for: The API key authenticates your requests to Google's Gemini API, enabling the extension to send YouTube video information and receive AI-generated summaries.

Is it free? Yes! Google provides free API keys for Gemini. You can get your free API key from Google AI Studio.

Is it safe? Your API key is stored securely in your browser's local storage and is only used to make requests to Google's API. It is never shared with third parties.
